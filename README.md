### Features

* 给麦麦加上了全新的自编程能力  
* 麦麦生成的代码在安全的DOCKER沙箱中运行，~~绝不会出现可乐视频中的高危操作(鞭尸)~~  
* 以及船新的百度搜索功能！麦麦也可以联网了 ~~你不会没配置百度api key吧~~  
* 还有一个戳一戳功能和一个~~没什么卵用的~~群签到功能
